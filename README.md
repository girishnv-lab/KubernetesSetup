# RHEL9k8onkind
This works for RHEL9 and this assumes that OM is already installed. 

Step 1 : 
cd /tmp

Step 2 :
git clone https://github.com/girishnv-lab/RHEL9k8onkind.git

Step 3 : 
sh RHEL9k8onkind/K8onKindRHEL9.sh

Step 4 : 

Enjoy....!!!!!
